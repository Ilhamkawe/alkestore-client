# Alkestore Frontend

## Technology
- Node.js
- Vue.js
- Tailwind

## Project setup

adjust backend url on .env like: 

```
VUE_APP_BACKEND_URL = "http://localhost:8000"
```

```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
